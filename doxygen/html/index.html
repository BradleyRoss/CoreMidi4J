CoreMidi4J

The purpose of the CoreMidi4J software is to provide a JNI interface to the CoreMidi library in Mac OS X.

References

  • Apple CoreMidi docoumentation is located at https://developer.apple.com/documentation/coremidi.
  • dOxygen documentation is located at http://www.doxygen.org.
  • See http://docs.oracle.com/javase/7/docs/technotes/guides/jni/spec/functions.html for some information on the C functions used by JNI.
